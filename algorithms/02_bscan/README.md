# B-Scan：一维线性扫查

本示例参考四阶碳钢厚度校准试块，沿 120 mm 直线生成 481 条模拟 RF A 扫，并排列成 B 扫侧视图。工作台支持点击或拖动选择扫查位置，联动查看对应的 A 扫，并可在包络与 RF 两种显示之间切换。

在仓库根目录运行：

```powershell
python algorithms/02_bscan/example.py
```

# B-Scan: One-Dimensional Linear Scan

This example references a four-step carbon-steel thickness calibration block, generating 481 simulated RF A-scans along a 120 mm line and arranging them into a B-scan side view. The workbench supports selecting a scan position by clicking or dragging to inspect the corresponding A-scan, and toggling between envelope and RF display modes.

Run it from the repository root:

```powershell
python algorithms/02_bscan/example.py
```
