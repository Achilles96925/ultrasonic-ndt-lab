# A-Scan：信号显示与处理

本示例模拟一条脉冲回波 A 扫，并以可交互工作台的形式显示完整处理流程：
原始信号、带通滤波、Hilbert 包络检测、时间增益补偿（TGC）和软件检测闸门。

在仓库根目录运行：

```powershell
python algorithms/01_ascan/example.py
```

# A-Scan: Signal Display and Processing

This example simulates a pulse-echo A-scan and presents the complete processing
pipeline in an interactive workbench: raw data, band-pass filtering, Hilbert envelope
detection, time-gain compensation (TGC), and a software inspection gate.

Run it from the repository root:

```powershell
python algorithms/01_ascan/example.py
```
