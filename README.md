# Ultrasonic NDT Lab

一个面向超声无损检测（NDT）成像算法的开源实验室。

本仓库将逐步收录可复现的实现、仿真、可视化和性能测试，覆盖从 A 扫
信号处理到相控阵与全矩阵采集成像的超声检测流程。

---

An open laboratory for ultrasonic non-destructive testing (NDT) imaging algorithms.

This repository will gradually collect reproducible implementations, simulations,
visualizations, and benchmarks for ultrasonic inspection workflows, from A-scan
signal processing to phased-array and full matrix capture imaging.
