# 超声成像算法实战——前言：从一条回波到一幅缺陷图像

最近准备把工作中接触到的一些超声检测算法记录下来。

内容会从最基础的A扫开始。

> 这里先简单说明一下后面会出现的几个术语。
>
> **A扫** 一条以时间和幅值表示的回波曲线。横坐标表示超声波的传播时间，也可以根据材料中的声速换算成声程或深度；纵坐标表示接收到的回波幅值。
>
> **B扫** 沿一个方向连续采集A扫后形成的二维截面图。
>
> **C扫** 是在一个平面内进行扫描，并从每个位置的A扫中提取指定深度范围内的特征值，最后得到缺陷在平面内的分布图。
>
> **S扫** 通常用于相控阵检测，通过改变声束角度，对一个截面范围进行扇形显示。
>
> **FMC** 是一种数据采集方式，会记录不同发射阵元和接收阵元之间的组合信号。
>
> **TFM** 是一种成像方法，它利用FMC采集到的数据，对成像区域中的每个像素进行延迟计算和信号叠加，从而得到图像。

先看一条回波信号是怎样产生的，以及它经过探头、采集端和数字处理之后，怎样变成软件里的一条曲线。

在这个基础上，再讨论如何把多条A扫组织成B扫和C扫，之后再进入相控阵、S 扫、FMC 和 TFM，逐步说明：

* 一个成像像素对应哪些采样数据；
* 声束的角度和焦点如何控制；
* 阵元之间的延迟如何计算；
* 数据量增加后如何优化。

这里不会一开始就把复杂公式全部堆出来，而是先从能看懂、能运行的简单模型开始，再逐步增加真实系统中的细节。实际超声检测中，采样率、增益、耦合、材料衰减和噪声，都会影响最终的图像。后面的内容会尽量把这些问题放回具体的数据和代码中讨论，而不是只停留在概念层面。

简单来说，这个系列想记录的是这样一条过程：
> 从探头接收到的一条回波开始，经过采集、处理、扫描和重建，最后得到一幅用于观察缺陷的图像。
